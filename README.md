# RQP Skateclub

RQP Skateclub website

### Prerequisites

* Composer
* Npm
* PHP 7.2
* Mysql database rqp_skateclub


### Launch localy

* composer install
* npm install into the public directory

and then run symfony server:start or php bin/console server:start
